# Community

RamRyder is currently a research prototype, and the current implementation has
been validated on a limited hardware platform. We welcome community support to
extend the system, report issues, and start technical discussions.

There are several ways to get involved:
1. Contribute new patches to support a broader range of hardware platforms.
2. Report bugs, issues, or unexpected behavior by opening an issue in the
   corresponding Github project under [ramryder-project](https://github.com/ramryder-project).
3. Reach out to `yaz093@ucsd.edu` for technical discussion and collaboration.
