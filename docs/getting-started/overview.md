# Getting Started Overview

RamRyder setup is organized into three components:

- **Resource Manager**: build and configure host-side memory resource management.
- **Virtual Machine Stack**: build QEMU integration and prepare/manage VM runtime.
- **Kernel**: build and install the RAMOS guest kernel.

Recommended build and setup sequence:

1. Complete [Resource Manager Setup](/getting-started/resource-manager-setup).
2. Complete [Virtual Machine Setup](/getting-started/virtual-machine-setup).
3. Complete [Kernel Installation](/getting-started/kernel-installation).

Use these three guides together for a full RamRyder bring-up.
