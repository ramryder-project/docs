# Overview
MemX OS is a Linux-based kernel extension that supports software-defined
memory in both guest virtual machines and host servers. It introduces a new
memory-channel abstraction to enable fine-grained page-to-channel allocation.

Currently, [RamRyder](/memx-ramryder/overview) uses the MemX OS kernel in
guest virtual machines. The host version is under development and will be
released soon. Before building the kernel, you should first launch a guest VM
with [RamRyder](/memx-ramryder/overview) and then follow the
[Quick Start](/memx-os/build) to build the MemX OS kernel for virtual
machines.
