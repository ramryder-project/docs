# memx-lab Docs Overview

`memx-lab` is the shared documentation home for memory-related projects, experiments, tooling, and technical notes.

This site serves one main job:

- A documentation portal across multiple related repositories

## What belongs here

- Cross-project concepts and architecture
- Shared quick starts
- Project index pages
- Roadmaps and design notes
- Contribution guides

## What stays in each project repo

- Project-specific API references
- Deep implementation notes tied to one codebase
- Release notes for a single component

That split keeps the main homepage focused while still giving users one clear place to start technically.
