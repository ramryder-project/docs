# Projects

Use this page as the index for every `MemX`-related repository.

## Suggested layout

### memx-core

Core runtime, engine, or library.

### memx-bench

Benchmarks and performance methodology.

### memx-examples

Examples, tutorials, and integration demos.

### memx-labs

Experimental work and prototypes.

As the organization grows, each item can link to:

- GitHub repository
- Short description
- Stability level
- Docs page
